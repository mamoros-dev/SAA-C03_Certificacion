# AWS Solutions Architect

+ AWS Solutions Architect es un rol profesional (y también una certificación) en el ecosistema de Amazon Web Services (AWS), la plataforma de nube más grande del mundo.

+ ¿Qué es?
    - Es un especialista en diseñar y planificar infraestructuras tecnológicas en la nube de AWS. Actúa como puente entre los requisitos del negocio y las soluciones técnicas.

+ ¿Qué hace?
    - Diseño de arquitecturas — Define cómo se estructuran los sistemas en la nube: servidores, bases de datos, redes, seguridad, etc.
    - Selección de servicios — Elige qué servicios de AWS usar (EC2, S3, RDS, Lambda, etc.) según las necesidades del proyecto.
    - Optimización de costos — Asegura que la infraestructura sea eficiente y no genere gastos innecesarios.
    - Seguridad y cumplimiento — Diseña sistemas que cumplan con estándares de seguridad y normativas.
    - Escalabilidad — Planifica que los sistemas puedan crecer sin fallar bajo mayor demanda.
    - Asesoría técnica — Guía a los equipos de desarrollo sobre las mejores prácticas en la nube.
    - Migración a la nube — Ayuda a empresas a mover sus sistemas locales (on-premise) a AWS.


+ La certificación
    - Existen dos niveles principales:
        - AWS Certified Solutions Architect – Associate → nivel intermedio
        - AWS Certified Solutions Architect – Professional → nivel avanzado
> Es una de las certificaciones tech más valoradas y mejor pagadas del mercado laboral actualmente.

```

TCP/IP, DNS, HTTP/S, VPNs, firewalls

```

Seguridad
```
IAM (Identity & Access Management) — muy valorado
Conceptos de Zero Trust, cifrado, gestión de secretos
```

💻 Infraestructura como Código (IaC)
```
Esto es casi obligatorio hoy en día:

```

🔄 DevOps y CI/CD
```
Docker y contenedores → esencial
Kubernetes (al menos conceptos, mejor si tienes la CKA)

```

📜 Scripting y automatización
```
Python → el lenguaje del cloud por excelencia

No necesitas ser desarrollador, pero sí automatizar tareas
```

📊 Observabilidad
```
CloudWatch (nativo AWS)
Conceptos de logging, métricas y alertas
Bonus: Grafana, Prometheus, Datadog
```
